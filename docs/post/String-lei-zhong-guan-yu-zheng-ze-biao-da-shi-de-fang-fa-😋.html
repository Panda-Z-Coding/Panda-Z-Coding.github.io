字符序列的分解

public String[] split(String regex); 
        //通过regex的正则表达式来分割当前String对象

Note

split()方法认为分隔符标记的左右都是单词，所以若左边是空字符，那么这个空字符算作第一个单词!

例如:String str = "公元1949年10月1日是中华人民共和国成立的日子"
当我们对其使用 regex = "//D+" 作为标记符时，数组里面的长度是4而不是3 !
